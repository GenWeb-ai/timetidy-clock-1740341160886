# Digital Clock

This project is a simple digital clock implemented using HTML, CSS, and JavaScript. It displays the current time and updates every second.

## Features

- Displays the current time in `HH:MM:SS` format.
- Automatically updates every second.
- Simple and clean user interface.

## Prerequisites

To run this project, you need a web browser that supports HTML5, CSS3, and JavaScript.

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/digital-clock.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd digital-clock
   ```

3. **Open the `index.html` file in your web browser:**

   You can do this by double-clicking the `index.html` file or by right-clicking and selecting "Open with" followed by your preferred web browser.

## Usage

Once you open the `index.html` file in your browser, the digital clock will be displayed in the center of the screen. The time will update automatically every second.

## Customization

- **Change Font Size:** You can adjust the font size of the clock by modifying the `font-size` property in the `styles.css` file under the `.clock` class.

- **Change Background Color:** To change the background color, modify the `background-color` property in the `styles.css` file under the `body` selector.
